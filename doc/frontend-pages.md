# Frontend Pages Plan

## 1. Home Page
Purpose:
- Introduce the Campus Coffee and Catering Services platform.
Main Elements:
- Navigation bar
- Welcome banner
- Featured menu items
- Quick access buttons

## 2. Login Page
Purpose:
- Allow users to log into the system.
Main Elements:
- Email input
- Password input
- Login button
- Register link

## 3. Menu Page
Purpose:
- Let users browse food, drinks, and catering packages.
Main Elements:
- Search bar
- Category filters
- Product cards
- Add to Cart button

## 4. Cart Page
Purpose:
- Display selected items before checkout.
Main Elements:
- Item list
- Quantity controls
- Total price
- Checkout button

## 5. Profile Page
Purpose:
- Manage user information and preferences.
Main Elements:
- Personal information
- Dietary preferences
- Order history
