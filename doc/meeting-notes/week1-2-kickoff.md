## Discussion Summary

### 1. Project Goal Confirmation
The team confirmed that the project is a centralized campus coffee and catering services web application for students, faculty, event organizers, and vendors.

### 2. Core Features Discussed
The team agreed that the first version should focus on:
- User registration and login
- Menu browsing
- Cart and ordering
- Catering booking
- Vendor/admin basic management

### 3. Technical Stack Confirmation
The team agreed to use the MERN stack:
- MongoDB
- Express.js
- React.js
- Node.js

Supporting tools may include:
- React Router
- Context API
- Axios
- Mongoose
- JWT
- Swagger or Apifox
- Figma
